# Privacy Policy

**Last updated: May 3, 2026**

## Overview

Meta Enricher is a local desktop application for Windows and macOS. It enriches photo metadata using AI. This policy describes what data the app accesses and how it is used.

## Data We Do Not Collect

Meta Enricher does not collect, transmit, or store any personal data on remote servers. The developer has no access to your data.

## Data Processed Locally

The app accesses the following data solely on your device:

- **Photos and image files** — to read and write EXIF/IPTC/XMP metadata (title, description, keywords, location, creator, copyright).
- **File system paths** — to locate your photo library folders.
- **App settings** — stored locally in your user profile (Ollama URL, model name, default creator/copyright fields). Never transmitted anywhere.

## AI Processing (Ollama)

Meta Enricher can use an AI vision model to generate metadata suggestions for your photos:

- **Local mode (default):** Photos are sent to Ollama running on your own machine (`localhost`). No data leaves your device.
- **Cloud mode (optional):** If you configure an Ollama Cloud API key, photo thumbnails are sent to Ollama's servers for processing. In this case, Ollama's own privacy policy applies. Your API key is stored locally on your device only.

## Third-Party Services

- **Nominatim (OpenStreetMap):** If GPS coordinates are present in a photo, the app may perform a reverse geocoding request to `nominatim.openstreetmap.org` to obtain a city/country name. Only the GPS coordinates are sent. No photos or personal data are transmitted. See [OpenStreetMap Privacy Policy](https://wiki.osmfoundation.org/wiki/Privacy_Policy).
- **Ollama Cloud (optional):** Only used if you explicitly enter an API key. See [Ollama Privacy Policy](https://ollama.com/privacy).

## Data Storage

All data generated by the app (settings, session notes, thumbnail cache) is stored locally in your system's application data folder and never leaves your device.

## Children's Privacy

This app is not directed at children under 13 and does not knowingly process data from children.

## Changes to This Policy

If this policy changes materially, the updated version will be posted here with a new date.
